Import Users from CSV

CSV File Format

Your CSV file must have the following columns (in this order):

  • email - User's email address (required, unique)
  • firstName - User's first name (required)
  • lastName - User's last name (required)
  • phone - Phone number (optional, can be empty)
  • isPrivacyEnabled - Privacy setting (true/false)

Example CSV:

email,firstName,lastName,phone,isPrivacyEnabled
newuser1@example.com,John,Smith,123456789,false
newuser2@example.com,Jane,Doe,987654321,true
newuser3@example.com,Bob,Johnson,,false
Note: All imported users will have the default password: password123
